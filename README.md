This project presents an interactive Power BI dashboard analyzing job vacancy data from PT XYZ in Indonesia.  
The analysis focuses on identifying job market trends based on job postings.

## Objectives
- Identify job market trends in Indonesia
- Analyze job vacancy distribution by role, location, and industry
- Provide an overview of current labor market conditions

## Tools and Technologies
- Power BI  
- Microsoft Excel  

## Dataset
The dataset consists of job vacancy postings collected from PT XYZ.  
The data includes job field, job position, number of vacancies, posting date, job location, salary range, and required education level.

## Key Findings
- Job vacancies are dominated by certain job fields such as Information Technology and Business.
- Job opportunities are concentrated in major cities.
- Most job postings require a minimum education level of D3 or S1.

## How to Use
1. Download the file `visual analysis.pbix`
2. Open the file using Power BI Desktop

This project was developed as a final assignment using Power BI.
